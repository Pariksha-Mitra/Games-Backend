# Games-Backend

A backend service for managing games and player interactions.

## Features
- Game management
- Player profiles
- Score tracking
- Multiplayer support

## Setup

1. Clone the repository
```bash
git clone https://github.com/Pariksha-Mitra/Games-Backend.git
cd Games-Backend
```

2. Install dependencies
```bash
npm install
```

3. Configure environment variables
```bash
cp .env.example .env
```

4. Start the server
```bash
npm run start
```
